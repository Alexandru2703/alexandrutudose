My Works

Here is a list of some of my projects.

Project 1

Project Description: Drone Flight Control

In this project, I developed custom code to control the flight of a drone. By writing and implementing algorithms, I enabled the drone to perform precise maneuvers, including takeoff, landing, and navigating through waypoints. The code integrated real-time data from various sensors to ensure stable and safe flight. In this project I've used the application "Mission Planner".

Project 2

Project Description: Topography Mapping

In this project, I harnessed the power of open-source tools and data to create detailed topography maps. By leveraging accessible resources and collaborating with the open-source community, I developed accurate and informative maps that showcased the terrain of various areas. This project not only emphasized my commitment to open-source principles but also demonstrated the ability to provide valuable geographic information for diverse applications, from environmental planning to recreational exploration. In this project I've used "QGis 3.16", "InkScape" "GRASS 7.8.4" and open source databases.

Project 3

Project Description: Laser-Cut Wood Creations with Machine Programming

This project was an exploration of crafting objects from wood using a 5W laser, but it went a step further by involving machine programming. Through careful design and coding, I harnessed the precision of laser technology to create intricate, custom-made wood objects. This project highlighted the synergy between artistic design and programming, resulting in a collection of finely detailed creations. It exemplifies how the fusion of creative design and meticulous machine programming can yield both functional and decorative pieces from natural materials, demonstrating the potential of technology to elevate craftsmanship.

Project 4

Project Description: Store Logo Design

This project involved the creation of a distinctive, registered logo for a store. Through a creative process that included market research, conceptualization, and graphic design, I developed a unique visual identity that encapsulates the brand's essence. The registered logo serves as a memorable and legally protected symbol that represents the store's values and products, enhancing its brand recognition and consumer trust.
